
XPINN code to solve the Poisson-Boltzmann equation.

$$\nabla^2 \phi_1 = -\frac{1}{\epsilon_1}\sum_k q_k\delta(\vb{x_k}) \:\:\:\: \vb{x} \in \Omega_1 $$
$$\nabla^2 \phi_2 = \kappa^2\phi_2 \:\:\:\: \vb{x} \in \Omega_2 $$
